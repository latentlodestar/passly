# Passly Mobile

React Native mobile app for Passly, built with [Expo](https://expo.dev) and [Expo Router](https://docs.expo.dev/router/introduction/) for file-based routing.

## Getting Started

1. Install dependencies:

   ```bash
   npm install
   ```

2. Set the API URL (defaults to `http://localhost:5192`):

   ```bash
   export EXPO_PUBLIC_API_URL=http://localhost:5192
   ```

3. Start the app:

   ```bash
   npx expo start
   ```

From the terminal, open the app in a:
- [Development build](https://docs.expo.dev/develop/development-builds/introduction/)
- [iOS Simulator](https://docs.expo.dev/workflow/ios-simulator/)
- [Android Emulator](https://docs.expo.dev/workflow/android-studio-emulator/)
- [Expo Go](https://expo.dev/go)

## Stack

- React Native + Expo
- Expo Router (file-based routing)
- Redux Toolkit Query (API communication)
- TypeScript
